# 04 — DHT11/DHT22 (อุณหภูมิ + ความชื้น)

## วงจร

| ขา sensor | ต่อกับ ESP32 |
|---|---|
| VCC | 3.3V |
| GND | GND |
| DATA | GPIO 15 |

## จุดสอน

- DHT เป็น **digital sensor**: ส่งค่ามาเป็นตัวเลขสำเร็จ (ต่างจาก LDR ที่เป็น analog)
- ห้ามอ่านถี่กว่าทุก 1–2 วินาที — อ่านเร็วไปจะ error
- `try/except OSError` คือการเขียนโปรแกรมแบบ "เผื่อพัง" — sensor จริงอ่านพลาดได้เสมอ
