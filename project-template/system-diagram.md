## Block Diagram

แก้ข้อความใน diagram นี้ให้ตรงกับระบบจริง:

```text
┌──────────────┐      ┌──────────────────┐      ┌──────────────┐
│    SENSE     │      │      THINK       │      │     ACT      │
│              │ ───▶ │                  │ ───▶ │              │
│ sensor: ____ │      │ if value > ____: │      │ LED: ______  │
│ ขา GPIO: ___ │      │     → WARNING    │      │ buzzer: ____ │
│ อ่านทุก: ___ │      │ if value > ____: │      │ web: _______ │
│              │      │     → ALERT      │      │              │
└──────────────┘      └──────────────────┘      └──────────────┘
                              │
                              ▼
                      ┌──────────────┐
                      │     LOG      │
                      │ data.csv     │
                      └──────────────┘
```
