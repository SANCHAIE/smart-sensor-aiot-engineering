## Sensor ที่ใช้ (ทีมเลือก/จับฉลาก 1 ตัวเป็นหลัก)

| Sensor | วัดอะไร | ขาต่อ | โค้ดตัวอย่าง |
|---|---|---|---|
| DHT11/DHT22 | อุณหภูมิ + ความชื้น | GPIO 15 | [`code-examples/04-dht/`](../code-examples/04-dht/) |
| LDR | ความเข้มแสง | GPIO 34 (ADC) | [`code-examples/05-ldr/`](../code-examples/05-ldr/) |
| HC-SR04 | ระยะทาง | Trig 26, Echo 27 | [`code-examples/06-ultrasonic/`](../code-examples/06-ultrasonic/) |

## โค้ดอ่านค่าพื้นฐาน (LDR)

```python
from machine import ADC, Pin
import time

ldr = ADC(Pin(34))
ldr.atten(ADC.ATTN_11DB)    # อ่านช่วง 0–3.3V → ค่า 0–4095

while True:
    value = ldr.read()
    print("Light level:", value)
    time.sleep(1)
```

## กิจกรรมสำคัญ: "ข้อมูล sensor ไม่นิ่ง"

ให้ทุกทีมทดลองแล้วสังเกต:
1. อ่านค่าโดย **ไม่แตะอะไรเลย** 10 ครั้ง — ค่าเท่ากันทุกครั้งไหม
2. เปลี่ยนสภาพแวดล้อม (เอามือบัง / เป่าลมร้อน / ขยับวัตถุ) แล้วดูค่าเปลี่ยน
3. คุยในทีม: ถ้าค่ากระโดดไปมา จะเชื่อค่าไหนดี (ปูทางไปเรื่อง average ครั้งหน้า)
