# 06 — HC-SR04 Ultrasonic (ระยะทาง)

## วงจร

| ขา sensor | ต่อกับ ESP32 |
|---|---|
| VCC | 5V (VIN) |
| GND | GND |
| Trig | GPIO 26 |
| Echo | GPIO 27 (ผ่าน voltage divider 1kΩ+2kΩ ถ้าเป็นรุ่น 5V) |

## หลักการ

ส่งคลื่นเสียงความถี่สูง → จับเวลาที่สะท้อนกลับ → ระยะ = เวลา × ความเร็วเสียง ÷ 2 (เหมือนค้างคาว)

## จุดสอน

- ทำไมต้องหาร 2 (เสียงเดินทางไป-กลับ)
- ทำไมต้องมี timeout (ถ้าไม่มีอะไรสะท้อน โปรแกรมจะค้างรอตลอดกาล)
- ช่วงวัดจริง ~2–400 ซม. พื้นผิวนุ่ม/เอียงสะท้อนแย่ — เอาไปออกแบบ test case ได้

## ใช้ทำโปรเจกต์อะไรได้

Smart Parking, Water Tank Monitor (วัดระดับน้ำจากด้านบน), เตือนสิ่งกีดขวาง
