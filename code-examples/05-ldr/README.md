# 05 — LDR (ความเข้มแสง, Analog)

| ไฟล์ | คำอธิบาย |
|---|---|
| `main.py` | อ่านค่าแสงพื้นฐาน |
| `smart_light.py` | ไฟติดเองเมื่อมืด + เฉลี่ยค่าก่อนตัดสิน |

## วงจร (Voltage Divider)

```text
3.3V ──[ LDR ]──●──[ 10kΩ ]── GND
                │
             GPIO 34
```

## จุดสอน

- LDR เป็น **analog sensor**: ได้ค่าต่อเนื่อง 0–4095 ไม่ใช่แค่ on/off
- ขา ADC ของ ESP32 ที่แนะนำ: 32–39 (ขา 34/35/36/39 เป็น input อย่างเดียว)
- ค่าจะแกว่งตลอด → เฉลี่ยหลายครั้งก่อนตัดสินใจ (ดู `smart_light.py`)
- threshold ของแต่ละห้อง/แต่ละช่วงเวลาไม่เท่ากัน → ต้องหาจากข้อมูลจริง (ครั้งที่ 5)
