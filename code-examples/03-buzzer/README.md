# 03 — Buzzer (Output เสียง)

`main.py` มี 2 ส่วน: เสียงเตือนแบบ on/off และเล่นโน้ตด้วย PWM

## วงจร

GPIO 18 → buzzer (+) ... (−) → GND

## Active vs Passive Buzzer

| ชนิด | วิธีใช้ | สังเกต |
|---|---|---|
| Active | `on()/off()` ก็ดัง | มีสติกเกอร์ปิดด้านบน, มีวงจรกำเนิดเสียงในตัว |
| Passive | ต้องป้อนความถี่ด้วย PWM | เห็นแผงวงจรด้านล่าง, เล่นโน้ตได้ |

## จุดสอน

- PWM = สัญญาณเปิด-ปิดเร็ว ๆ — ความถี่กำหนดระดับเสียง, duty กำหนดความดัง
- เสียงเตือนที่ต่างกัน (สั้น/ยาว/ถี่) สื่อความหมายต่างกันได้ — ใช้ออกแบบระดับ ALERT ของ final project
