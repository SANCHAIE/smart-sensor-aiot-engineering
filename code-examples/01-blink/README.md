# 01 — Blink LED

โปรแกรมแรกของทุกทีม: ทำให้ไฟ LED บนบอร์ดกระพริบ

| ไฟล์ | คำอธิบาย |
|---|---|
| `main.py` | Blink พื้นฐาน 0.5 วินาที |
| `challenge_sos.py` | เฉลย Challenge SOS (ครูใช้ดูแนวทาง) |

## วงจร

ไม่ต้องต่ออะไร — LED บนบอร์ดอยู่ที่ GPIO 2 (บอร์ดบางรุ่นอาจเป็นขาอื่น ถ้าไม่ติดให้ลองต่อ LED ภายนอก)

## จุดสอน

- `Pin(2, Pin.OUT)` = ประกาศขา output
- `while True:` = วนตลอด — นี่คือโครงของโปรแกรมฝังตัวเกือบทุกตัว
- ไฟล์ชื่อ `main.py` บนบอร์ดจะรันเองอัตโนมัติเมื่อเสียบไฟ
