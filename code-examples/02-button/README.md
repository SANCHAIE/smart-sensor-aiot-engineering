# 02 — Button (Input)

| ไฟล์ | คำอธิบาย |
|---|---|
| `main.py` | กดปุ่มแล้วไฟติด |
| `challenge_modes.py` | เฉลย Challenge เปลี่ยนโหมด 3 โหมด พร้อม debounce |

## วงจร

- Button: GPIO 4 → ปุ่ม → GND
- LED: GPIO 5 → ตัวต้านทาน 220Ω → LED ขายาว (+) ... ขาสั้น → GND
- Buzzer (สำหรับ challenge): GPIO 18 → buzzer (+) ... (−) → GND

## จุดสอน

- `Pin.PULL_UP`: ปล่อยปุ่ม = อ่านได้ 1, กดปุ่ม = 0 (เด็กมักงงว่าทำไมกลับด้าน)
- **Debounce**: ปุ่มกลไกเด้ง ทำให้กด 1 ครั้งเครื่องเห็นหลายครั้ง → ต้องเว้นเวลาขั้นต่ำระหว่างการกด
- `time.ticks_ms()` ใช้จับเวลาแบบไม่ block ทั้งโปรแกรม (ดีกว่า `sleep` ยาว ๆ)
